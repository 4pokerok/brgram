# brgram Roadmap

## Phase 0 - Foundation (now)

- Monorepo setup
- Web app + API
- PostgreSQL schema
- JWT auth
- Realtime text channels
- MVP voice channels

## Phase 1 - Core Messenger

- Direct messages and private groups
- Channel permissions (owner/admin/member)
- Message edits, deletes, reply threads
- File upload pipeline
- Basic moderation and anti-spam

## Phase 2 - Production Hardening

- Redis adapter for realtime scale
- SFU-based voice infrastructure
- Push notifications (WebPush, APNs, FCM)
- Observability and alerts
- Backup and disaster recovery policy

## Phase 3 - Cross-Platform Clients

- React Native apps (iOS/Android)
- Tauri/Electron desktop apps (macOS/Windows)
- Unified authentication and sync
- Offline cache and reconnect strategy

## Phase 4 - Telegram-Class Features

- Bots and mini-apps platform
- Message translation and smart search
- Communities and public channels
- Voice/video calls and conference controls
